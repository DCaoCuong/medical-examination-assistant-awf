# Phase 10: Cloud Deployment

Status: ⬜ Pending

## Objective
Đưa ứng dụng lên môi trường Production (Vercel + Supabase Cloud) để sử dụng thực tế.

## Tasks:
- [ ] **Environment Setup:** Chuẩn bị tệp `.env.production` và cấu hình biến môi trường trên Vercel.
- [ ] **Database Migration:** Chuyển đổi dữ liệu từ Local sang Supabase Cloud Project.
- [ ] **Build Optimization:** Chạy `npm run build` cục bộ để kiểm tra các lỗi Type-check hoặc Linting.
- [ ] **Vercel Deployment:** Kết nối GitHub Repository và thực hiện deploy lên Vercel.
- [ ] **Domain & SSL:** Cấu hình tên miền (nếu có) và kiểm tra chứng chỉ bảo mật SSL.
- [ ] **Final Smoke Test:** Kiểm tra các tính năng chính (Ghi âm, AI Analysis, Export PDF) trên trang live.

## Output:
- Link ứng dụng chính thức (URL: https://medical-assistant-awf.vercel.app - *ví dụ*).
- Hệ thống hoạt động ổn định trên internet.
